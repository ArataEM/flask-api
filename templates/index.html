List of api methods:

  • "/api" - returns this page (GET, 200)
  • "/api/students" - returns list of all students (GET, 200)
  • "/api/students/get/<id>" - returns student by id (GET, 200)
  • "/api/students/add" - creates new student (POST, 201)
  • "/api/students/modify/<id>" - changes student's data (PATCH, 200)
  • "/api/students/change/<id>" - changes all student's data (PUT, 200)
  • "/api/students/delete/<id>" - removes student (DELETE, 204)
